ABOUT US

We developed a website for analysis of data of geotechical engineering

This project is basically about the analysis of different soil samples from different places of India to produce a map API which shows all the data of the experiment which performs on the different soils. This project requires some coding knowledge as it contains Map APIs and web development .

The locations from which we take soil samples are:
1. Rohtak,Haryana -YC
2. Rewari ,Haryana- YF
3. Guwahati,Assam-BP
4. Barpeta,Assam-BG
5. Tihu,Nalbari district,Assam-RS and PC
